# its-travel-pack

## How to use

#### clone

```
$ git clone git@github.com:suzuki/its-travel-pack.git
```

#### composer

```
$ composer install
```

#### get font

```
$ cd its-travel-pack
$ ./bin/getFont.php
```

#### run

```
$ php app.php
```

#### open PDF (using OS X)

```
$ open pdf/export/travelpack_reserve.pdf
```
